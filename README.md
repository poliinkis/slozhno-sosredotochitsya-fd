«Сложно сосредоточиться»

Проект интерактивного лендинга с галереей, секциями контента и переключением тем. Скрипты для переключения светлой, тёмной и авто-темы уже предоставлены обучающей платформой, задача — стилизовать страницу и сделать адаптивную верстку.

Технологии: HTML, CSS (Flexbox, Grid, переменные и медиазапросы), базовый JavaScript

Особенности проекта:

Mobile first подход и адаптивная «резиновая» верстка.
Переключение тем: тёмная, светлая, авто.
Статичная и интерактивная верстка секций с заголовками, карточками и ссылками.
Фиксированное фоновое изображение и декоративные элементы через псевдоэлементы.
Использование CSS-переменных для цветов, шрифтов и отступов.
Адаптивная типографика с использованием clamp().

Запуск: открыть index.html в браузере.
